💾 What is Walrus?
Walrus is a decentralized storage and data availability protocol built on Sui. It allows you to store large blobs of data such as images, files, game state, and metadata in a trustless, cost-efficient way, directly accessible from your Sui smart contracts and dApps.

For this hackathon, your project must use Walrus as part of its core functionality, not just as an add-on.
